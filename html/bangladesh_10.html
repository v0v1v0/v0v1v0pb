map_division R Documentation

Banlgadesh administrative level 1 shapefile

Description

A shapefile containing level 1 administrative boundaries

Usage

map_division

Format

A shapefile with 5 variables:

Division

division (admin level 1) names

ADM1_PCODE

admin level 1 codes

Country

country (admin level 0) name

ADM0_PCODE

admin level 0 codes

geometry

MULTIPOLYGON for administrative areas

Source

Bangladesh Bureau of Statistics
