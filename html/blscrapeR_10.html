naics R Documentation

Dataset containing NIACS codes for industry lookups.

Description

Built-in dataset to look up industries by NAICS code. To access the data directly, issue the command naics).

  • industry_code: Industry code

  • industry_title: Industry title

Usage

data(naics)

Format

A data frame with 2469 rows and 2 variables
