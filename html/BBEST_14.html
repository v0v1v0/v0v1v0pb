fix.merge R Documentation

Merge .fix files

Description

fix.merge merges several .fix files into a specified file in a form suitable for PDFgetN.

Usage

fix.merge(outfile, infile1, infile2, ...)

Arguments

outfile

character, the filename for saving the data.
infile1, infile2, ...

files to merge.
