ocskew R Documentation

Octile Skewness

Description

A function to calculate an estimate of the octile skewness from a set of data.

Usage

ocskew(x)

Arguments

x

A vector of numeric values.

Value

The octile skewness.

Author(s)

Richard Murray Lark <murray.lark@nottingham.ac.uk>

Examples

x<-evapotranspiration$`ET(mm)`
ocskew(x)
