rna R Documentation

Build a RNA vector

Description

rna() build a RNA vector from a character vector.

Usage

rna(...)

Arguments

...

characters to turn into RNA. Can be a set of name-value pairs.

Value

a vector of class bioseq_rna

See Also

Other classes: aa(), dna()

Examples


rna("AGGUGC", "UUCGA")

rna(Seq_1 = "AGGUGC", Seq_2 = "UUCGA")

x <- c("AGGTGC", "TTCGA")
rna(x)
