state_fips R Documentation

Dataset with the lat. / long. of county FIPS codes used for mapping.

Description

Built-in dataset for use with the bls_map function. To access the data directly, issue the command datastate_fips).

  • fips_state: FIPS code for state

  • state_abb: State abbreviation

  • state: State name

  • gnisid: Geographic Names Information System ID

Usage

data(state_fips)

Format

A data frame with 57 rows and 4 variables
