linkingMatrix R Documentation

Linking matrix

Description

Linking numbers between all pairs of strands of a braid.

Usage

linkingMatrix(braid)

Arguments

braid

a braid object

Value

A matrix.

See Also

strandLinking to get the linking number between two strands of the braid.

Examples

braid <- mkBraid(4, c(2, -3, 3))
linkingMatrix(braid)
