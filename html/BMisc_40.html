rhs.vars R Documentation

Right-hand Side Variables

Description

Take a formula and return a vector of the variables on the right hand side

Usage

rhs.vars(formla)

Arguments

formla

a formula

Value

vector of variable names

Examples

ff <- yvar ~ x1 + x2
rhs.vars(ff)

ff <- y ~ x1 + I(x1^2)
rhs.vars(ff)
