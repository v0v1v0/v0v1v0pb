last_year R Documentation

Last (most recent) year of an fhx object

Description

Last (most recent) year of an fhx object

Usage

last_year(x)

Arguments

x

An fhx object.

Value

The maximum or last year of series in x. NA will be returned if NA is in x$year.

See Also
