untreatedFuncMat R Documentation

Untreated Samples with 4 Different Treatments (Functional)

Description

a matrix containing the signal intensity for functional markers measured using CyTOF on different cell types, after stimulation with BCR/FcR-XL, PMA/Ionomycin and vanadate. See Bodenmiller et al 2012 for details.

Usage

untreatedFuncMat

Format

a matrix with 15792 rows and 14 variables:

  • pStat1

  • pSlp76

  • pBtk

  • pPlcg2

  • pErk

  • pLat

  • pS6

  • pNFkB

  • pp38

  • pStat5

  • pAkt

  • pSHP2

  • pZap70

  • pStat3

Source

https://reports.cytobank.org/105/v2/
