Rename sets

Description

Rename sets.

Usage

name_sets(object) <- value

Arguments

object

A TidySet object.
value

A character with the new names for the sets.

Value

A TidySet object.

    sets = c(rep("a", 5), "b"),
    elements = letters[seq_len(6)],
    fuzzy = runif(6)
