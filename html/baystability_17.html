g_eff R Documentation

Genotype Effects

Description

Calcuates Genotype Effects

Usage

## Default S3 method:
g_eff(.data, .y, .gen, .env)

Arguments

.data

data.frame
.y

Response Variable
.gen

Genotypes Factor
.env

Environment Factor

Value

Genotype Effects
