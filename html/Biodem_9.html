mtx.exp R Documentation

Calculates the n-th power of a matrix

Description

Calculates the n-th power of a matrix.

Usage

mtx.exp(X, n)

Arguments

X

a square matrix

n

the exponential value

Details

This function calculates (efficiently!) the n-th power of a matrix.

Value

Takes a matrix and returns a matrix.
