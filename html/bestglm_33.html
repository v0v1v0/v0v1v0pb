sphereX R Documentation

Sphere Data Matrix

Description

The data matrix is scaled and sphered so it is orthonormal. The Cholesky decomposition is used.

Usage

sphereX(X)

Arguments

X

X rectangular data matrix

Value

sphered matrix

Author(s)

A. I. McLeod

See Also

scale, NNPredict

Examples

data(longley)
longley.x <- data.matrix(longley[, 1:6])
sphereX(longley.x)
