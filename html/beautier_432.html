is_mcmc R Documentation

Determine if the object is a valid MCMC

Description

Determine if the object is a valid MCMC

Usage

is_mcmc(x)

Arguments

x

an object, to be determined if it is a valid MCMC

Value

TRUE if x is a valid MCMC, FALSE otherwise

Author(s)

Rich√®l J.C. Bilderbeek

See Also

Use create_mcmc to create an MCMC

Examples

if (is_on_ci()) {
  check_empty_beautier_folder()

  # Returns TRUE
  is_mcmc(create_mcmc())
  is_mcmc(create_ns_mcmc())

  # Returns FALSE
  is_mcmc("nonsense")
  is_mcmc(NULL)
  is_mcmc(NA)
  is_mcmc("")
  is_mcmc(c())

  check_empty_beautier_folder()
}
