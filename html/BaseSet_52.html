getGAF R Documentation

Read a GAF file

Description

Read a GO Annotation File (GAF) formatted file

Usage

getGAF(x)

Arguments

x

A file in GAF format

Value

A TidySet object

References

The format is defined here.

See Also

Other IO functions: getGMT(), getOBO()

Examples

gafFile <- system.file(
    package = "BaseSet", "extdata",
    "go_human_rna_valid_subset.gaf"
)
gs <- getGAF(gafFile)
head(gs)
