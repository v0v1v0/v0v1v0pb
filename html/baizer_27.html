dx_tb R Documentation

diagnosis a tibble for character NA, NULL, all T/F column, blank in cell

Description

diagnosis a tibble for character NA, NULL, all T/F column, blank in cell

Usage

dx_tb(x)

Arguments

x

tibble

Value

list

Examples

x <- tibble::tibble(
  c1 = c("NA", NA, "a", "b"),
  c2 = c("c", "d", "e", "NULL"),
  c3 = c("T", "F", "F", "T"),
  c4 = c("T", "F", "F", NA),
  c5 = c("", " ", "\t", "\n")
)

dx_tb(x)
