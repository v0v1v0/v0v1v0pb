Predict R Documentation

Classify a new observation point

Description

Classify a new observation point

Usage

Predict(Point, Fit)

Arguments

Point

A new observation
Fit

A BEST object

Value

The predicted class

Examples

n <- 500
Data <- BESTree::Data[1:n,]
NewPoint <- BESTree::Data[n+1,]
d <- ncol(Data)-1
VA <- ForgeVA(d,1,0,0,0)
Size <- 50
Fit <- BESTree::BEST(Data,Size,VA)
BESTree::Predict(NewPoint[1:d],Fit)
