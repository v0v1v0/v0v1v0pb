setAttribute R Documentation

A wrapper for attr(x, which) = y.

Description

A wrapper for attr(x, which) = y.

Usage

setAttribute(x, which, value)

Arguments

x

[any]
Your object.
which

[character(1)]
Name of the attribute to set
value

[ANY]
Value for the attribute.

Value

Changed object x.

Examples

setAttribute(list(), "foo", 1)
