nlist R Documentation

Named lists

Description

Functions to construct, coerce and check for named lists.

Usage

nlist(...)

as.nlist(x, ...)

is.nlist(x)

Arguments

...

Named objects.
x

Object to be coerced or tested.

Value

A named list.

Examples

x <- nlist(x = 2, y = c("a", "b"))
is.nlist(x)
