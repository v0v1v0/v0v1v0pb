tbflt R Documentation

create a tbflt object to save filter conditions

Description

tbflt() can save a series of filter conditions, and support logical operating among conditions

Usage

tbflt(x = expression(), .env = NULL)

Arguments

x

any expression
.env

environment

Value

tbflt

Examples

c1 <- tbflt(cut == "Fair")

c2 <- tbflt(x > 8)

!c1

c1 | c2

c1 & c2
