qqnormy R Documentation

Quantile generator

Description

A Q-Q value generator.

Usage

qqnormy(y)

Arguments

y

data sample

Details

This is an equivalent to qqnorm, but returning sorted values. See qqnorm.

Value

y

vector of quantile values.
