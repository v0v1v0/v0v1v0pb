MergeMCMC R Documentation

Merge MCMC

Description

Merge two or more MCMC simulations

Usage

MergeMCMC(pat, project.dir = "Results/", data.sets)

Arguments

pat

pattern to select MCMC chain from
project.dir

define where to save data, Default: 'Results/'
data.sets

data sets to combine

Value

Merged MCMC chains
