Truncated R Documentation

Truncates a Distribution

Description

Truncates a Julia Distribution between 'lower' and 'upper'.

Usage

Truncated(dist, lower, upper)

Arguments

dist

A Julia Distribution created using Gamma, InverseGamma ...
lower

lower bound
upper

upper bound

Value

see Gamma
