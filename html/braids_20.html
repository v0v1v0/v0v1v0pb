tau R Documentation

Inner automorphism

Description

The inner automorphism defined by \tau X = \Delta^{-1} X \Delta, where \Delta is the positive half-twist; it sends each generator \sigma_j to \sigma_{n-j}.

Usage

tau(braid)

Arguments

braid

a braid object

Value

A braid object.

Examples

braid <- mkBraid(4, c(2, -3, 3))
tau(braid)
