dna R Documentation

Build a DNA vector

Description

dna() build a DNA vector from a character vector.

Usage

dna(...)

Arguments

...

characters to turn into DNA. Can be a set of name-value pairs.

Value

a vector of class bioseq_dna

See Also

Other classes: aa(), rna()

Examples


dna("AGGTGC", "TTCGA")

dna(Seq_1 = "AGGTGC", Seq_2 = "TTCGA")

x <- c("AGGTGC", "TTCGA")
dna(x)
