BreastCancer R Documentation

Breast Cancer Data

Description

Cleaned breast cancer data from TCGA. Three datasets: primary, normal and meta.

Usage

data(BreastCancer)

Value

None

References

TCGA(2012) Comprehensive molecular portraits of human breast tumours. Nature,490(7418),61-70.

Examples

 data(BreastCancer)
 head(normal)
 head(primary)
 head(meta)
