optimal R Documentation

Retrieve the index of the optimal probability threshold.

Description

Retrieve the index of the optimal probability threshold.

Usage

optimal(x)

Arguments

x

An object of class "bayefdr".

Value

The integer index of the optimal probability threshold.

Examples

 probs <- runif(100)
 e <- efdr_search(probs, target_efdr = 0.1)
 optimal(e)
 e[optimal(e), ]
