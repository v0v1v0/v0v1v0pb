EyxbivgeomRoy R Documentation

Conditional moment

Description

Conditional moment of Y given X=x for Roy's bivariate geomtric model

Usage

EyxbivgeomRoy(theta1, theta2, theta3, x)

Arguments

theta1

paramater \theta_1

theta2

paramater \theta_2

theta3

paramater \theta_3

x

value of the conditioning variable X

Value

Value of the conditional moment of Y given X=x

Author(s)

Alessandro Barbiero

References

Roy, D. (1993) Reliability measures in the discrete bivariate set-up and related characterization results for a bivariate geometric distribution, Journal of Multivariate Analysis 46(2), 362-373.

See Also

FyxbivgeomRoy

Examples

theta1 <- 0.5
theta2 <- 0.7
theta3 <- 0.9
EyxbivgeomRoy(theta1, theta2, theta3, 2)
