Description

Set up a volatility prior

Usage

set_SV(
  priors,
  mu = 0,
  phi = 0.5,
  sigma = 1,
  nu = Inf,
  rho = 0,
  beta = 0,
  latent0 = 0
)

Arguments

priors

Prior settings from specify_priors.
mu, phi, sigma, nu, rho, beta, latent0

Numerics with starting values for the respective parameter.

Value

Returns a list with priors and settings.
