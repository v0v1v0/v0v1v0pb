Trim R Documentation

Trim

Description

remove excess whitespace from string

Usage

Trim(s, multi = TRUE)

Arguments

s

string
multi

logical, indicating whether or not to remove excess whitespace between characters, Default: TRUE

Examples

 Trim("             Trimmed      string")
 # [1] "Trimmed string"
 Trim("             Trimmed      string", FALSE)
 # [1] "Trimmed      string"
