read_x3p R Documentation

Read an x3p file as an R Data Frame

Description

Read an x3p file as an R Data Frame

Usage

read_x3p(path, profiley = TRUE)

Arguments

path

The file path to the x3p file
profiley

If TRUE, rotate the matrix (if necessary) to ensure a profile is taken across y

Examples
