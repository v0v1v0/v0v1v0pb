split_line R Documentation

split line

Description

takes a line for a SNP file and splits into parts.

Usage

split_line(x)

Arguments

x

line from SNP file

Value

tibble with 6 columns.

Examples

split_line("1_14.570829090394763     1        0.000000              14 A X")
split_line("rs3094315	1	0.0	752566	G	A")
