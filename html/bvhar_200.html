bound_bvhar R Documentation

Setting Empirical Bayes Optimization Bounds

Description

This function sets lower and upper bounds for set_bvar(), set_bvhar(), or set_weight_bvhar().

Usage

bound_bvhar(
  init_spec = set_bvhar(),
  lower_spec = set_bvhar(),
  upper_spec = set_bvhar()
