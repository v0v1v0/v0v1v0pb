bcmixed R Documentation

bcmixed: A package for mixed effect models with the Box-Cox transformations.

Description

The bcmixed package provides two categories of important functions: bcmarg and bcmmrm. The bcmarg function provides inferences on the marginal model of the mixed effect model with the Box-Cox transformation and the bcmmrm provides inferences on the model median differences between treatment groups for longitudinal randomized clinical trials. These statistical methods are proposed by Maruo et al. (2017, https://doi.org/10.1002/sim.7279).
