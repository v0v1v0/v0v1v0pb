ar1Solver R Documentation

Compute AR1 correlation coefficient

Description

Computes value for AR1 correlation coefficient for use in p_adjust

Usage

ar1Solver(t)

Arguments

t

A numeric vector of t-statistics

Value

Estimated AR1 correlation coefficient

See Also

p_adjust

Examples

t <- rt(1e3, df = 1)
rho <- ar1Solver(t)
