add_column R Documentation

Add column

Description

Add column to a slot of the TidySet object.

Usage

add_column(object, slot, columns)

## S4 method for signature 'TidySet,character'
add_column(object, slot, columns)

Arguments

object

A TidySet object.
slot

A TidySet slot.
columns

The columns to add.

Value

A TidySet object.

Methods (by class)

  • add_column(object = TidySet, slot = character): Add a column to any slot

Examples

relations <- data.frame(
    sets = c(rep("a", 5), "b"),
    elements = letters[seq_len(6)],
    fuzzy = runif(6)
)
TS <- tidySet(relations)
add_column(TS, "relations", data.frame(well = c(
    "GOOD", "BAD", "WORSE",
    "UGLY", "FOE", "HEY"
)))
