Description

Configures a project to use {box.lsp}

Usage

use_box_lsp(file_path = ".Rprofile")

Arguments

file_path

File name to append {box.lsp} configuration lines.

Value

Writes configuration lines to file_path.

Examples

if (interactive()) {
  use_box_lsp()
}
