Acc R Documentation

Computes the proportion of matching terms in two vectors of the same length. Used to compute the accuracy for prediction on test set.

Description

Computes the proportion of matching terms in two vectors of the same length. Used to compute the accuracy for prediction on test set.

Usage

Acc(Vec1, Vec2)

Arguments

Vec1

A vector of labels
Vec2

Another vector of labels

Value

Percentage of identical labels (accuracy)

Examples

Vec1 <- c(1,1,2,3,1)
Vec2 <- c(1,2,2,3,1)
Acc(Vec1,Vec2)
