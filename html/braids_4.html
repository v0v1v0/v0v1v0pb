braidPermutation R Documentation

Braid permutation

Description

Returns the left-to-right permutation associated to a braid.

Usage

braidPermutation(braid)

Arguments

braid

a braid object (e.g. created with mkBraid)

Value

A permutation.

Examples

braid <- mkBraid(4, c(2, -3, 3))
braidPermutation(braid)
