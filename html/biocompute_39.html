Parse Biocompute Object From JSON File to R Object

Description

Parse Biocompute Object From JSON File to R Object

Usage

read_bco(x, ...)

Arguments

x

BioCompute Object .json file
...

Additional parameters for fromJSON.

Value

A list of class bco

Examples

bco <- tempfile(fileext = ".json")
bco <- generate_example("HCV1a") %>%
  convert_json() %>%
  export_json(bco)
bco %>% read_bco()
