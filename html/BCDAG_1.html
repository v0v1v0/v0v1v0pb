as_graphNEL R Documentation

Transform adjacency matrix into graphNEL object

Description

Function to transform an adjacency matrix into a graphNEL object.

Usage

as_graphNEL(DAG)

Arguments

DAG

Adjacency matrix of a DAG

Value

A graphNEL object

Examples

# Randomly generate DAG
q <- 4; w = 0.2
set.seed(123)
DAG <- rDAG(q,w)
as_graphNEL(DAG)
