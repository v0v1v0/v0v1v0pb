randomUnitary R Documentation

Random unitary

Description

Returns a square complex matrix sampled from the Haar random unitary distribution.

Usage

randomUnitary(size)

Arguments

size

dimension of matrix

Value

A square complex matrix.

Examples

m <- 25 # size of matrix (m x m)
set.seed(7)
U <- randomUnitary(m)
#  
n <- 5 # First n columns
A <- U[,1:n]
