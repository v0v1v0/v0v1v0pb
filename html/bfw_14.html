GammaDist R Documentation

Gamma Distribution

Description

compute gamma distribution (shape and rate) from mode and standard deviation

Usage

GammaDist(mode, sd)

Arguments

mode

mode from data
sd

standard deviation from data

Examples

 GammaDist(1,0.5)
 # $shape
 # [1] 5.828427
 # $rate
 # [1] 4.828427
