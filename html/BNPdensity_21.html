dhalft R Documentation

Density half Student-t

Description

Computes the density.

Usage

dhalft(x, df = 1, mean = 0, sd = 1)

Details

For internal use

Examples


## The function is currently defined as
function(x, df = 1, mean = 0, sd = 1) {
  ifelse(x < 0, 0, 1) * dt_(x, df, mean, sd) / (1 - pt_(
    0, df,
    mean, sd
  ))
}
