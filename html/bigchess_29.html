uci_read R Documentation

Read current stdout from chess engine

Description

Read current stdout from chess engine

Usage

uci_read(engine)

Arguments

engine

engine object

Value

engine object

Examples


# Linux (make sure you have executable permission):
engine_path <- "./stockfish_10_x64"
# Windows
# engine_path <- "./stockfish_10_x64.exe"
e <- uci_engine(engine_path)
e <- uci_read(e)
e$temp
uci_quit(e)
