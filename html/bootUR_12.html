find_nonmissing_subsample R Documentation

Find Non-Missing Subsamples

Description

Find Non-Missing Subsamples

Usage

find_nonmissing_subsample(X)

Arguments

X

A (TxN)-matrix of N time series with T observations. Data may also be in a time series format (e.g. ts, zoo or xts) or data frame. Assumes a prior check on missing values in-sample has been done.

Value

A list with the following components

range

(2xN)-dimensional matrix containing the first and last non-missing observation in each column of X.
all_equal

Logical value indicating whether all series have the same non-missing indices.
