inverseBraid R Documentation

Inverse braid

Description

The inverse of a braid (without performing reduction).

Usage

inverseBraid(braid)

Arguments

braid

a braid object

Value

A braid object.

Examples

braid <- mkBraid(4, c(2, -3, 3))
ibraid <- inverseBraid(braid)
composeTwoBraids(braid, ibraid)
