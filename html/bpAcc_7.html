Methods functions in bpAcc

Description

The function ProbAccept returns an object of (S4) class 'ProbAcceptance', a list with elements required by methods defined for ProbAcceptance objects. See below.

Methods

Methods for objects of class 'ProbAcceptance'.

show

To display the probability of tolerable error in a friendly version.

plot

To plot the estimated probabiliy of tolerable error based on simulated data, as described in ProbAccept
