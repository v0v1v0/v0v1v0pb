area_titles R Documentation

Dataset containing FIPS codes for counties, states and MSAs.

Description

Built-in dataset to look up areas by FIPS code. Includes recent and historical FIPS codes. To access the data directly, issue the command area_titles).

  • area_fips: FIPS code

  • area_title: Area name

Usage

data(area_titles)

Format

A data frame with 4723 rows and 2 variables
