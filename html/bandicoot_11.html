%eq% R Documentation

The equals to operator

Description

The equals to operator

Usage

x %eq% y

Arguments

x

Object.
y

Another object.

Value

A Boolean value.

Examples

AGE <- new_class(class_name = "AGE")
age <- AGE$instantiate
register_method(AGE, ..init.. = function(current) self$current = current)
register_method(AGE, ..eq.. = function(y) self$current == y$current)

patrick <- age(33)
james <- age(33)
patrick %eq% james
