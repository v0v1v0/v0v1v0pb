reg_join R Documentation

join the matched parts into string

Description

join the matched parts into string

Usage

reg_join(x, pattern, sep = "")

Arguments

x

character
pattern

regex pattern
sep

separator

Value

character

Examples

reg_join(c("A_12.B", "C_3.23:2"), "[A-Za-z]+")

reg_join(c("A_12.B", "C_3.23:2"), "\\w+")

reg_join(c("A_12.B", "C_3.23:2"), "\\d+", sep = ",")

reg_join(c("A_12.B", "C_3.23:2"), "\\d", sep = ",")
