Get error messages of jobs.

Description

Get error messages of jobs.

Usage

getErrorMessages(reg, ids)

Arguments

reg

[Registry]
Registry.
ids

[integer]
Ids of jobs. Default is all jobs with errors.

Value

[character]. Error messages for jobs as character vector
NA if job has terminated successfully.
