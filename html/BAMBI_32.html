Description

Fitting univariate wrapped normal mixtures using MCMC

Usage

fit_wnormmix(...)

Arguments

...

arguments (other than model) passed to fit_angmix

Details

Wrapper for fit_angmix with model = "wnorm".

Examples

# illustration only - more iterations needed for convergence
