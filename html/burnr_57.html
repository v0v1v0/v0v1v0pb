write_fhx R Documentation

Write an fhx object to a new FHX2 file

Description

Write an fhx object to a new FHX2 file

Usage

write_fhx(x, fname = "")

Arguments

x

An fhx object.
fname

Output filename.

See Also

  • write.csv() to write a CSV file. Also works on fhx objects.

  • read_fhx() to read an FHX2 file.

Examples

write_fhx(lgr2, "afile.fhx")
