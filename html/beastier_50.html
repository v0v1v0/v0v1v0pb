  fasta_filename,
  taxa_name_ext = ""

Arguments

n_taxa

The number of taxa
sequence_length

a DNA sequence length, in base pairs
fasta_filename

a FASTA filename.
taxa_name_ext

the extension of the taxa names

Value

Nothing, creates a FASTA file
