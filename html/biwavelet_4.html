arrow R Documentation

Helper function for phase.plot (not exported)

Description

Helper function for phase.plot (not exported)

Usage

arrow(x, y, l = 0.1, w = 0.3 * l, alpha, col = "black")

Arguments

x

X-coordinate of the arrow.
y

Y-coordinate of the arrow.
l

Length of the arrow.
w

Width of the arrow.
alpha

Angle of the arrow in radians (0 .. 2*pi).
col

Color of the arrow.

Examples

plot.new()
arrow(0,0, alpha = 0)
