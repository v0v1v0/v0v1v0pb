is.empty R Documentation

Test emptyness

Description

These methods test if an object x is empty.

Usage

is.empty(x)

## Default S3 method:
is.empty(x)

## S3 method for class 'data.frame'
is.empty(x)

Arguments

x

An object to be tested.

Value

TRUE if x is empty, FALSE otherwise.

See Also

as.empty in this package.

Examples

is.empty(4)
is.empty(c())
is.empty(new.env())
is.empty(character(0))
is.empty(list())
is.empty(integer(0))
is.empty(data.frame())
