big_attach R Documentation

Attach a Filebacked Big Matrix

Description

Attach a Filebacked Big Matrix

Usage

big_attach(rdsfile)

big_attachExtdata()

Arguments

rdsfile

Path to a ".rds" file.

Value

The FBM object stored in the rdsfile.

Examples

# temporary FBM
X <- FBM(10, 10)$save()

rdsfile <- sub_bk(X$backingfile, ".rds")
X2 <- big_attach(rdsfile)

all.equal(X[], X2[])
