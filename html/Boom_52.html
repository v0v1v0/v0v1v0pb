is.even R Documentation

Check whether a number is even or odd.

Description

Check whether a number is even or odd.

Usage

IsEven(x)
IsOdd(x)

Arguments

x

An integer or vector of integers.

Value

Logical indicating whether the argument is even (or odd).

Author(s)

Steven L. Scott

Examples

IsEven(2) ## TRUE
IsOdd(2)  ## FALSE
