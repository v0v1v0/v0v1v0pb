full2simple R Documentation

Convert encoding from full to simple format

Description

Converts an encoding from the full format to the simple format.

Usage

full2simple(x)

Arguments

x

encoding.

Examples

aa1 = list(`1` = c("g", "a", "p", "v", "m", "l", "i"), 
           `2` = c("k", "h"), 
           `3` = c("d", "e"), 
           `4` = c("f", "r", "w", "y", "s", "t", "c", "n", "q"))
full2simple(aa1)
