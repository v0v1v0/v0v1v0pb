Adjacency

Description

Are two elements connected ?

Usage

## S3 method for class 'TidySet'
adjacency(object)

adjacency_element(object)

adjacency_set(object)

## S3 method for class 'TidySet'
adjacency(object)

Arguments

object

A TidySet object

Value

A square matrix, 1 if two nodes are connected, 0 otherwise.

See Also

incidence()

Examples

x <- list("SET1" = letters[1:5], "SET2" = LETTERS[3:7])
a <- tidySet(x)
adjacency_element(a)
adjacency_set(a)
