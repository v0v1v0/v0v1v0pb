bs4Jumbotron R Documentation

BS4 jumbotron for AdminLTE3

Description

Create a jumbotron

Usage

bs4Jumbotron(
  ...,
  title = NULL,
  lead = NULL,
  href = NULL,
  btnName = "More",
  status = c("primary", "warning", "danger", "info", "success")
)

jumbotron(
  ...,
  title = NULL,
  lead = NULL,
  href = NULL,
  btnName = "More",
  status = c("primary", "warning", "danger", "info", "success")
)

Arguments

...

Any content.
title

Jumbotron title.
lead

Jumbotron lead.
href

Jumbrotron external link.
btnName

Jumbotron button name.
status

Jumbotron background color. "primary", "success", "warning", "danger" or "info".

Author(s)

David Granjon, dgranjon@ymail.com
