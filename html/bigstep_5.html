bic R Documentation

BIC

Description

Calculate BIC (Bayesian Information Criterion).

Usage

bic(loglik, k, n)

Arguments

loglik

A numeric, the log-likelihood.
k

An integer >= 0, the number of selected variables.
n

An integer > 0, the number of observations.

Value

A number, a value of BIC.

Examples

bic(10, 5, 100)
