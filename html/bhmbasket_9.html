invLogit R Documentation

invLogit

Description

This function returns the inverse logit of the input argument.

Usage

invLogit(theta)

Arguments

theta

A numeric

Details

This function is an alias for 'stats::binomial()$linkinv'

Value

Inverse logit of theta

Author(s)

Stephan Wojciekowski

See Also

family

Examples

invLogit(logit(0.3))
invLogit(c(-Inf, 0, Inf))
