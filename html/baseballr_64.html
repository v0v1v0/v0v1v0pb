load_umpire_ids R Documentation

Download a data frame of all umpires and their mlbamids for games since 2008

Description

Download a data frame of all umpires and their mlbamids for games since 2008

Usage

load_umpire_ids()

Value

Function returns a tibble with the following columns:

  • id

  • position

  • name

  • game_pk

  • game_date

Examples


  try(load_umpire_ids())
