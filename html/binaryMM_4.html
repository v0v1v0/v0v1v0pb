Description

Compute Gauss-Hermite quadrature rule

Usage

get.GH(q, scale_abscissa = sqrt(2), scale_weight = 1/sqrt(pi))

Arguments

q

Order of the Gauss-Hermite quadrature rule to compute
scale_abscissa

Fixed number
scale_weight

Fixed number

Value

A list with the following components:

z

Nodes

w

Quadrature Weights
