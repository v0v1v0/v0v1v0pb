map_district R Documentation

Banlgadesh administrative level 2 shapefile

Description

A shapefile containing level 2 administrative boundaries

Usage

map_district

Format

A shapefile with 7 variables:

District

district (admin level 2) names

ADM2_PCODE

admin level 2 codes

Division

division (admin level 1) names

ADM1_PCODE

admin level 1 codes

Country

country (admin level 0) name

ADM0_PCODE

admin level 0 codes

geometry

MULTIPOLYGON for administrative areas

Source

Bangladesh Bureau of Statistics
