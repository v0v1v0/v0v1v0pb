sim_geno

Description

Simulated geno file of eigenstrat format

Usage

sim_geno(n_ind, n_snp, filename)

Arguments

n_ind

number of individuals
n_snp

number of SNPs
filename

filename of export

Value

NULL exports a file

Examples

## Not run: 
sim_geno(10, 5, "geno.txt")

## End(Not run)
