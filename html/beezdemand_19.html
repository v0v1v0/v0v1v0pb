Pull

Description

Pull vector from data frame

Usage

pull(x, y)

Arguments

x

A data frame
y

Name of column

Details

Pulls a single vector from a data frame. Good to use with dplyr. From http://stackoverflow.com/questions/21618423/extract-a-dplyr-tbl-column-as-a-vector

Value

Vector
