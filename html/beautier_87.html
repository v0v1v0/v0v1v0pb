check_tree_prior R Documentation

Check if the tree prior is a valid tree prior

Description

Calls stop if the tree priors are invalid

Usage

check_tree_prior(tree_prior)

Arguments

tree_prior

a tree priors, as returned by create_tree_prior

Value

nothing
