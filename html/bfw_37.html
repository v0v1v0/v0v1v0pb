SingleString R Documentation

Single String

Description

determine whether input is a single string

Usage

SingleString(x)

Arguments

x

string

Value

true or false

Examples

A <- "This is a single string"
SingleString(A)
# [1] TRUE
is.character(A)
# [1] TRUE
B <- c("This is a vector" , "containing two strings")
SingleString(B)
# [1] FALSE
is.character(B)
# [1] TRUE
