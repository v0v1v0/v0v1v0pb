unfortify_x3p R Documentation

Convert a data frame into an x3p file

Description

Convert a data frame into an x3p file

Usage

unfortify_x3p(df)

Arguments

df

A data frame produced by fortify_x3p

Value

An x3p object

Examples

data(br411)
br411_fort <- fortify_x3p(br411)
br411_unfort <- unfortify_x3p(br411_fort)
identical(br411_unfort, br411)
