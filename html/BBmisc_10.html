catf R Documentation

Wrapper for cat and sprintf.

Description

A simple wrapper for cat(sprintf(...)).

Usage

catf(..., file = "", append = FALSE, newline = TRUE)

Arguments

...

[any]
See sprintf.
file

[character(1)]
See cat. Default is “”.
append

[logical(1)]
See cat. Default is FALSE.
newline

[logical(1)]
Append newline at the end? Default is TRUE.

Value

Nothing.

Examples

msg = "a message."
catf("This is %s", msg)
