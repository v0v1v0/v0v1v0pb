mlb_people R Documentation

Find Biographical Information for MLB Players

Description

Find Biographical Information for MLB Players

Usage

mlb_people(person_ids = NULL)

Arguments

person_ids

MLBAMIDs for players of interest. Multiple IDs should be provided in a vector separated by a comma.
