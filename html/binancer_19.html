binance_query R Documentation

Request the Binance API

Description

Request the Binance API

Usage

binance_query(
  endpoint,
  method = c("GET", "POST", "PUT", "DELETE"),
  params = list(),
  body = NULL,
  sign = FALSE,
  retry = method == "GET",
  content_as = "parsed"
)

Arguments

endpoint

string
method

HTTP request method
params

URL parameters provided as a list
body

body of the request
sign

if signature required
retry

allow retrying the query on failure
content_as

parameter to httr::content

Value

R object
