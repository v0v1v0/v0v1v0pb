APSsystem R Documentation

APSsystem Bayesian Network

Description

An ERP data quality assessment framework for the implementation of an APS system using Bayesian networks.

Format

A discrete Bayesian network for data quality assessment. Probabilities were given within the referenced paper. The vertices are:

QPlanDeliveryTime

(Complete, Incomplete);

QSetupTime

(Complete, Incomplete);;

PlanDeliveryTime

(Complete, Incomplete);

SetupTime

(Complete, Incomplete);

NNTransactionData

(Complete, Incomplete);

NNMasterData

(Complete, Incomplete);

NNValues

(High, Low);

Completeness

(High, Low);

Consistency

(High, Low);

DataQuality

(High, Low);

Value

An object of class bn.fit. Refer to the documentation of bnlearn for details.
