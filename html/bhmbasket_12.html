logit R Documentation

logit

Description

This function returns the logit of the input argument.

Usage

logit(p)

Arguments

p

A numeric in (0, 1)

Details

This function is an alias for 'stats::binomial()$linkfun'

Value

logit of p

Author(s)

Stephan Wojciekowski

See Also

family

Examples

logit(invLogit(0.3))
logit(c(0, 0.5, 1))
