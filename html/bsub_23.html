plot_dependency R Documentation

Plot the job dependency tree

Description

Plot the job dependency tree

Usage

plot_dependency(job_id, job_tb = NULL)

Arguments

job_id

A job ID.
job_tb

A table from bjobs. Optional.

Value

No value is returned.

Examples

## Not run: 
job1 = random_job()
job2 = random_job()
job3 = random_job(dependency = c(job1, job2))
plot_dependency(job3)

## End(Not run)
