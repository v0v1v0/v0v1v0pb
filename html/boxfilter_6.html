store R Documentation

Store data

Description

Stores original and filtered data.

Usage

store(object)

Arguments

object

Must be of class "boxclip", resulting from boxclip.

Value

returns nothing

Author(s)

Thomas Ruf (thomas.p.ruf@me.com)

Examples

data ("wb_month.RData")

x=wb_month$x
y=wb_month$hr

myclip=boxclip(x,y,clipit=0.25,width=15)
summary(myclip)
store(myclip)
