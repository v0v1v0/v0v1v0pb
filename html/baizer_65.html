number_fun_wrapper R Documentation

wrapper of the functions to process number string with prefix and suffix

Description

wrapper of the functions to process number string with prefix and suffix

Usage

number_fun_wrapper(
  x,
  fun = ~.x,
  prefix_ext = NULL,
  suffix_ext = NULL,
  verbose = FALSE
)

Arguments

x

number string vector with prefix and suffix
fun

process function
prefix_ext

prefix extension
suffix_ext

suffix extension
verbose

print more details

Value

processed number with prefix and suffix

Examples

number_fun_wrapper(">=2.134%", function(x) round(x, 2))
