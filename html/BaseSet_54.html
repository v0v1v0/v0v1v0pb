getOBO R Documentation

Read an OBO file

Description

Read an Open Biological and Biomedical Ontologies (OBO) formatted file

Usage

getOBO(x)

Arguments

x

Path to a file in OBO format.

Value

A TidySet object.

References

The format is described here

See Also

Other IO functions: getGAF(), getGMT()

Examples

oboFile <- system.file(
    package = "BaseSet", "extdata",
    "go-basic_subset.obo"
)
gs <- getOBO(oboFile)
head(gs)
