permutationBraid R Documentation

Permutation braid

Description

Makes a permutation braid from a permutation.

Usage

permutationBraid(perm)

Arguments

perm

a permutation

Value

A braid object.

Examples

perm <- c(3, 1, 4, 2)
braid <- permutationBraid(perm)
isPermutationBraid(braid)
braidPermutation(braid)
