boost2 R Documentation

Some Shortcuts

Description

Some simple shortcuts to model fitting engines.

Usage

## BayesX.
bayesx2(...)

## Gradient boosting.
boost2(...)

## Lasso.
lasso2(...)

Arguments

...

Arguments passed to bamlss and predict.bamlss.

Value

A bamlss object.

See Also

bamlss, boost, lasso, BayesX

Examples

## Not run: ## Simulate data.
set.seed(123)
d <- GAMart()

## Estimate model.
f <- num ~ s(x1) + s(x2) + s(x3) + s(lon,lat)

## Boosted model.
b <- boost2(f, data = d)

## Plot estimated effects.
plot(b)

## End(Not run)
