untreatedPhenoMat R Documentation

Untreated Samples with 4 Different Treatments (Phenotypic)

Description

a matrix containing the signal intensity for phenotypic markers measured using CyTOF on different cell types, after stimulation with BCR/FcR-XL, PMA/Ionomycin and vanadate. See Bodenmiller et al 2012 for details.

Usage

untreatedPhenoMat

Format

a matrix with 15792 rows and 9 variables:

  • CD20

  • IgM

  • CD4

  • CD33

  • HLA-DR

  • CD14

  • CD7

  • CD3

  • CD123

Source

https://reports.cytobank.org/105/v2/
