Description

Fitting bivariate wrapped normal model mixtures using MCMC

Usage

fit_wnorm2mix(...)

Arguments

...

arguments (other than model) passed to fit_angmix

Details

Wrapper for fit_angmix with model = "wnorm2".

Examples
